## Description

Headless road cycling race simulation engine. Pure TS, no framework dependencies (framework-agnostic by design). Access with Rest API.

- terrain impact
- fatigue system
- group / peloton logic (decision model)
- sprint phase
- team tactics AI (strategy behavior)
- breakaways & attacks
- weather system
- crashes
- raider personalities (behavior archetypes)

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
npm run build       # compile TypeScript
npm run start:dev   # watch mode
npm run lint        # eslint
```

## Run tests

```bash
npm run test        # jest
npm run test:watch  # jest --watch
```
